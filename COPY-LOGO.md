# Manual Logo Setup Instructions

The logo file needs to be copied to the `public` folder. Please run this command in your terminal:

```bash
cd /Users/tunckahveci/Desktop/2026CoderWorld-Prj3/Content
mkdir -p public
cp ../Assets/ardictech-logo.png public/ardictech-logo.png
```

Or manually:
1. Navigate to `/Users/tunckahveci/Desktop/2026CoderWorld-Prj3/Content`
2. Create a folder named `public` if it doesn't exist
3. Copy `ardictech-logo.png` from `/Users/tunckahveci/Desktop/2026CoderWorld-Prj3/Assets/` to the `public` folder

After copying, restart the dev server:
```bash
npm run dev
```
